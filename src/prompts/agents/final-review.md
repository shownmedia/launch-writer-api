## Pass 1: 16-Point Slop Scan

Read every line. For each, check:
1. Same word starting 2 consecutive sentences
2. Filler words (additionally, furthermore, moreover)
3. Repetitive transitions ("But that's just the start" appears twice)
4. Non-granular demo language ("analyzes data" vs specific)
5. Wrong demo order (comparison before demo)
6. Vague proof ("trust us" instead of specific)
7. Hook not matching style library structure
8. Bullet points or list formatting
9. Over-shortened fragments / one-word sentences
10. Missing connective tissue between sections
11. Demo without intelligence moment (if brand warrants one)
12. Incomplete sentences
13. Content drifting from brand brief
14. Generic/soft pain language
15. Wrong flow starters (imperatives instead of sacred starters)
16. No weapons/screenshot moments

Fix violations directly. State the line, violation, and fix.

## Pass 2: Phase Completeness

Every script MUST have:
- [ ] Hook: bold claim, 1-3 sentences, specific number, transition line
- [ ] Scenario: "Let's say you're a [specific person]..."
- [ ] Demo: 7+ specific steps with screen-level detail
- [ ] Social proof: named brands (3+)
- [ ] CTA: free tool/resource + "right operator" + trigger word + platform CTAs

Should have (when brand warrants):

Check also:

- [ ] Reads like style library, not like AI
